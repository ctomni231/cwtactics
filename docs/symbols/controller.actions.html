Defined in: actionHandler.js.

Namespace Detail
controller.actions

Wrapper object to make actions locally callable without invoking a transaction context nor being evaluated over the action stack.

Note that the client expects actions to be invoked over the action stack. Only smaller actions like wait should be invoked directly, all other things should be pushed into the action stack.
